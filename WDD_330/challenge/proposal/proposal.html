Purpose

Create a build calculator for the popular game, League of Legends (LoL).

Background

In a LoL match, players earn gold through performing various actions. They use gold to buy items from the item shop, granting their character (champion) various upgrades and abilities. The items a player owns and the resulting stats their champion has are collectively termed their build.

To date, I have not found a good resource for simulating builds and comparing them against other possible builds. The best way to do this currently is to actually launch the game in practice mode and equip different items. While this method works, it is somewhat cumbersome, and obviously not suitable for "theory-crafting" away from the player's gaming machine.

Proposal

My proposal is to connect with the Riot Games API and create a build calculator where players can choose a character and different items to see the stats they grant.

Base Features

  1. ✅ Import champions via API
  2. ✅ Import items via API
  3. ✅ User can select a champion from a list
  4. ✅ User can select 1-6 items for their build from a list (builds max out at 6 items in game)
  5. Display item details
  6. ✅ Display build stats
  7. ✅ User can delete an item from the build and add a new one
  8. ✅ User can switch champions
  9. ✅ Responsive design

Stretch Features

  1. User can edit champion level
  2. User can search for item by name
  3. User can search for champion by name

Audience

League of Legends players

Data Sources

Riot Games API - Provided by the developer of League of Legends. Provides current data on item and character stats.
